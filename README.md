# Confluence Page Export

Export Confluence pages using REST API as `.doc` for given ID while preserving the hierarchy.

## Requirements
- Python 3.10
- `Requests` package

## Reference

- https://developer.atlassian.com/server/confluence/confluence-server-rest-api/
